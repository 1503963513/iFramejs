面板 1 - 基础信息 ▼

这是面板 1 的内容区域。当展开时，页面高度会增加。

高度自动同步功能会自动将新高度发送给父页面。

面板 2 - 高级功能 ▼

这是面板 2 的内容区域。

iFrame.js 使用 ResizeObserver API 来监听 DOM 变化。

当检测到高度变化时，会自动通过 postMessage 发送给父页面。

面板 3 - 使用说明 ▼

父页面设置：

iframeApp.enableAutoResize();

子页面设置：

childApp.startAutoResizer({ offset: 0 });
